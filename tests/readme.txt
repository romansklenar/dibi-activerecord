Tests Setup
===========

To ensure proper functionality of tests run PHPUnit with switch
	--bootstrap /path/to/bootstrap.php

If you are using Netbeans with PHP and PHPUnit support you can set this by:
	- in Project Window (CTRL + 1) right click on your project
	- choose "Properties"
	- in Project Properties dialog choose on the left side PHPUnit
	- in PHPUnit Configuration check "Use Bootstrap" and select bootstrap file in this folder
